Privacy Policy for MessageBox

Last updated: 28 april 2025

I am a solo independent front-end developer maintaining MessageBox. This privacy policy explains what information I collect and how it is used.

Information Collection

When you send a message through MessageBox, I collect:

  • Message content and timestamp
  • Device information:
    • Browser type and version
    • Operating system
    • Screen resolution
    • Window size
    • Device memory and CPU information
    • Touch capability
    • Network connection type
  • Location information:
    • Time zone
    • Language settings

Purpose of Collection

This information is collected for:

  • Message moderation and abuse prevention
  • Technical troubleshooting
  • Understanding user behavior to improve the application
  • Ensuring proper functionality across different devices

Data Storage and Security

All data is stored on Firebase's secure servers. Only I, as the administrator, have access to the detailed information. Messages are automatically deleted after reaching the maximum limit to maintain performance and data minimization.

Your Rights

You have the right to:

  • Opt out of data collection (though this may limit functionality)
  • Request deletion of your messages
  • Ask what information is stored about you

Contact

For any privacy-related questions or concerns, you can reach me at: [Your Contact Info]

Changes to This Policy

This privacy policy may be updated occasionally. Users will be notified of any significant changes through the application.
